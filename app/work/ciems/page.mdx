# Stage PFA chez CIEMS

En tant qu'élève ingénieur en première année, j'ai effectué mon stage ouvrier chez CIEMS, une startup de consulting à Rabat. La mission consistait à développer une preuve de concept (PoC) pour une entreprise d'assurance : une application intelligente capable de suggérer à ses clients des options d'assurance non souscrites, puis de communiquer avec eux via emails et SMS personnalisés.

---

## Le Projet : Recommandation Intelligente d'Assurances

L'objectif était de créer un système qui analyse les profils clients et propose de manière proactive des produits d'assurance complémentaires adaptés à leurs besoins.

### Stack Technique

Pour réaliser ce projet, mon équipe et moi avons choisi :

- **Django** : Framework backend robuste pour gérer la logique métier et les données clients
- **Twilio API** : Envoi de SMS et gestion des communications
- **OpenAI API** : Personnalisation automatique des messages selon le profil de chaque client
- **PostgreSQL** : Base de données pour stocker les informations clients et les interactions

---

## Fonctionnalités Développées

### 1. Système de Recommandation

L'application analyse le portefeuille d'assurances existant de chaque client et identifie les couvertures manquantes potentiellement pertinentes. L'algorithme prend en compte :

- Le profil démographique du client
- Les produits déjà souscrits
- L'historique des sinistres
- Les patterns de comportement similaires

### 2. Communication Personnalisée

Grâce à l'intégration de l'API OpenAI, chaque message envoyé est adapté au contexte du client :

- Ton et style ajustés selon le profil
- Références aux produits existants du client
- Arguments de vente personnalisés
- Traitement multilingue (français/arabe)

### 3. Gestion Multi-Canal

Les clients peuvent être contactés via :

- **Email** : Pour les propositions détaillées avec documentation
- **SMS** : Pour les rappels courts et les alertes
- **Dashboard** : Interface de suivi pour les agents d'assurance

---

## Défis Techniques Rencontrés



**Problème** : Manipulation de données sensibles (informations personnelles, santé, finances).

**Solution** : 
- Chiffrement des données en transit et au repos
- Anonymisation pour les tests
- Logs d'audit pour toute consultation de données
- Mécanisme de consentement explicite avant envoi

## L'IA au Service du Business

Ce projet illustre parfaitement comment l'intelligence artificielle peut créer de la valeur concrète :

**Avant** : Les agents envoyaient des emails génériques à des listes segmentées manuellement, avec un taux de réponse < 5%.

**Après** : Un système intelligent analyse chaque profil, génère des propositions pertinentes et rédige des messages personnalisés, multipliant l'engagement par 3+.
